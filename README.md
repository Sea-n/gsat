# 學測五選四
自 108 年開始，學測最多採計四科，來看看您心目中的科系怎麼算吧

## 使用方法
1. 開啟 [學測五選四](https://sean.cat/gsat) 網站
2. 選擇**個人申請**或是**繁星推薦**
3. 輸入想查詢的校系
4. 點選科目名稱按鈕，切換狀態
5. 覺得實用就點右下角分享出去吧 :smiley:
